# zhvue

> 仿移动安卓版知乎（个人学习项目）

vue2 + vuex2 + vue-router + stylus

现阶段数据均为mock数据（data.json）

由于知乎功能很多，现阶段仅完成几个小功能。并且由于暂未开发后端，个别交互逻辑稍有欠缺，后期会不断完善。

## 使用方法

``` bash
npm install

npm run dev

npm run build

npm run build --report
```
